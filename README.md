# RHttpClient
動作確認はしてますがおかしいところがありそうな HttpClient です。

Maven にも置いてます。

# Maven
- Repository
```XML
  <repositories>
    <repository>
      <id>net.simplyrin</id>
      <name>api</name>
      <url>https://api.simplyrin.net/maven/</url>
    </repository>
  </repositories>
```

- Dependency
```XML
  <dependencies>
    <dependency>
      <groupId>net.simplyrin.httpclient</groupId>
      <artifactId>RHttpClient</artifactId>
      <version>1.1-SNAPSHOT</version>
    </dependency>
  </dependencies>
```
